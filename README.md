# CodeAlpha Stock Portfolio Tracker 

A Python-based stock portfolio tracker designed to help users manage stock investments and calculate total portfolio value through a simple command-line interface.

---

##  Project Overview
This project allows users to add stock details, view their portfolio, and calculate total investment value using Python.

It demonstrates the implementation of dictionaries, loops, conditional statements, and financial calculation logic.

---

##  Features
- Add stock information
- View portfolio details
- Calculate total investment value
- Beginner-friendly command-line interface
- Simple portfolio management system

---

##  Concepts Used
- Python dictionaries
- Loops and conditions
- User input handling
- Basic financial calculations

---

##  Tech Stack
- Python 3

---

## How to Run

1. Open terminal in project folder  
2. Run the following command:

```bash
python portfolio.py
```
